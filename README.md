1. **Scene 1: Welcome Intro Stage**

## 🚀 Getting Started

To run and view the project locally:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation
1. Clone the repository coordinates.
2. Open your shell in the project root directory.
3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server
Run the local Vite development server:
```bash
npm run dev
```
Open the local URL displayed in your terminal (typically `http://localhost:5173`) in a web browser.

### Production Build
Compile and optimize the codebase for web deployments:
```bash
npm run build
```
This outputs optimized, static assets inside the `dist/` directory.
